Excel File Processor

Upload an Excel file to process it according to the specified business rules. The tool will categorize data by region and technology type.

How to use:

  • Upload an Excel file with the required columns (DPdeniro, S_SP, S_Total, etc.)
  • The processor will filter data based on CFS Cluster and Barangay groups
  • Files will be categorized into South, Central, and North regions
  • Spare files will exclude VDSL, ADSL, and ADSL/VDSL technologies
  • A separate DSL file will be created for those technologies
  • All files will include coordinate information
